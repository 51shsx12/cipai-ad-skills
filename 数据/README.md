# 投放数据文件夹

把巨量引擎导出的 CSV 报表放到这个文件夹，然后说「分析数据」即可自动读取最新文件。

## CSV 导出方式

巨量引擎后台 → 报表 → 导出报表 → 下载 CSV

## 文件名格式

随意命名即可，按日期命名推荐：
- `2025-06-22.csv`
- `日报_0622.csv`

## 必需字段

CSV 中至少包含以下列（中文列名会自动识别）：

| 必需列 | 对应字段 |
|--------|----------|
| 日期 | 哪天 |
| 展现量 / 展示次数 | 广告曝光 |
| 点击量 / 点击次数 | 广告点击 |
| 消耗 / 花费 | 花了多少钱 |
| 转化量 / 转化数 | 表单提交数（回传） |
| 深度转化量 | 通信退订（取消订购） |

## 示例数据

| 日期 | 展现量 | 点击量 | 消耗 | 转化量 | 深度转化量 |
|------|--------|--------|------|--------|------------|
| 2025-06-21 | 12000 | 180 | 58 | 8 | 2 |
| 2025-06-22 | 9500 | 130 | 42 | 5 | 1 |
